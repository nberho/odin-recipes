Burritos Recipe

Description

Here's a recipe for making super spicy burritos. You grab some tortillas and put a lot of stuff inside them, like meat and salad. Then add some chilly. But before eating, roll them like a fat cigar. Then eat them.

Ingredientes

  • Tortillas
  • Meats
  • Salads
  • Spicy sauces

Steps

  • Grab the tortilla
  • Open it
  • Put everything inside it
  • Add sauce on top
  • Roll it like a super huge cigar
  • Eat it
